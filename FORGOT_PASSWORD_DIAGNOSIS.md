## 🔧 **故障排除**

### **🐛 可能的問題**

#### **1. Gmail 安全設定**
**症狀：** 郵件發送失敗
**解決：**
- 檢查 Gmail 密碼是否正確
- 確認 Gmail 是否啟用兩步驟驗證
- 使用應用程式密碼而非一般密碼

#### **2. 網路連線**
**症狀：** API 請求超時
**解決：**
- 檢查網路連線
- 確認防火牆設定
- 檢查 SMTP 連接埠（587）

#### **3. 瀏覽器相容性**
**症狀：** JavaScript 錯誤
**解決：**
- 使用現代瀏覽器（Chrome, Firefox, Safari）
- 檢查瀏覽器控制台錯誤
- 確認 JavaScript 已啟用

#### **4. 驗證碼時效**
**症狀：** 驗證碼無效
**解決：**
- 驗證碼有效期限為10分鐘
- 確認在時效內使用
- 重新發送新的驗證碼
